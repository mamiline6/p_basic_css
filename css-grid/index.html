flexbox 指定一覧

1. display: grid;

display: grid;を指定した要素がグリッドになります。また、ラッパーのグリッド内のすべての要素がグリッドアイテムになります（グリッドアイテム内の要素はグリッドアイテムにならない点に注意）。

グリッドアイテム1
グリッドアイテム2
グリッドアイテム3
グリッドアイテム4
グリッドアイテム5
グリッドアイテム6
                        
            <!-- html -->
            <div class="grid-wrapper">
                <div class="grid-items">グリッドアイテム1</div>
                <div class="grid-items">グリッドアイテム2</div>
                <div class="grid-items">グリッドアイテム3</div>
                <div class="grid-items">グリッドアイテム4</div>
                <div class="grid-items">グリッドアイテム5</div>
                <div class="grid-items">グリッドアイテム6</div>
            </div>
                        
                        
            /* css */
            .grid-wrapper {
                display: grid;
            }

2. grid ラッパー向け：グリッドの行と列を定義。grid-template-columns/rowプロパティ

横方向（行）を定義する grid-template-columns プロパティ。縦方向（列）を定義する grid-template-rows プロパティ

2-1. grid ラッパーで gridアイテムの行と列を定義する

gridアイテムが3行2列に並びます

グリッドアイテム1
グリッドアイテム2
グリッドアイテム3
グリッドアイテム4
グリッドアイテム5
グリッドアイテム6
                        
            /* css */
            .grid-wrapper {
                display: grid;
                grid-template-columns: 1fr 1fr 1fr; /* 行数と幅 */
                grid-template-rows: 50px 50px;      /* 列数と幅 */
            }

2-2. repeat記法でトラック（グリッドアイテム）を列挙する

gridアイテムが3列（repeat記法）

グリッドアイテム1
グリッドアイテム2
グリッドアイテム3
グリッドアイテム4
グリッドアイテム5
グリッドアイテム6
                        
            /* css */
            .grid-wrapper {
                display: grid;
                grid-template-columns: repeat(3, 1fr); /* 1fr 1fr 1frと同じ */
            }

2-3. repeat記法でトラック（グリッドアイテム）を列挙する

10のトラックの内、1fr トラックに 2fr トラックが続くパターンを5回繰り返す（repeat記法）

グリッドアイテム1
グリッドアイテム2
グリッドアイテム3
グリッドアイテム4
グリッドアイテム5
グリッドアイテム6

グリッドアイテム9
グリッドアイテム10
                        
            /* css */
            .grid-wrapper {
                display: grid;
                grid-template-columns: repeat(5, 1fr 2fr);
            }
