flexbox 指定一覧

1. display: grid;

display: grid;を指定した要素がグリッドになります。また、ラッパーのグリッド内のすべての要素がグリッドアイテムになります（グリッドアイテム内の要素はグリッドアイテムにならない点に注意）。

グリッドアイテム2

グリッドアイテム4
グリッドアイテム5
グリッドアイテム6
                        
            <!-- html -->
            <div class="grid-wrapper">
                <div class="grid-items">グリッドアイテム1</div>
                <div class="grid-items">グリッドアイテム2</div>
                <div class="grid-items">グリッドアイテム3</div>
                <div class="grid-items">グリッドアイテム4</div>
                <div class="grid-items">グリッドアイテム5</div>
                <div class="grid-items">グリッドアイテム6</div>
            </div>
                        
                        
            /* css */
            .grid-wrapper {
                display: grid;
            }

2. grid ラッパー向け：グリッドの行と列を定義。grid-template-columns/rowプロパティ

横方向（行）を定義する grid-template-columns プロパティ。縦方向（列）を定義する grid-template-rows プロパティ

2-1. grid ラッパーで gridアイテムの行と列を定義する

gridアイテムが3行2列に並びます

                grid-template-rows: 50px 50px;      /* 列数と幅 */
