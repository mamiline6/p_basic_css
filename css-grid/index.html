CSS Grid 指定一覧

1. display: grid;

display: grid;を指定した要素がグリッドになります。また、ラッパーのグリッド内のすべての要素がグリッドアイテムになります（グリッドアイテム内の要素はグリッドアイテムにならない点に注意）。

グリッドアイテム1
グリッドアイテム2
グリッドアイテム3
グリッドアイテム4
グリッドアイテム5
グリッドアイテム6
                            
                <!-- html -->
                <div class="grid-wrapper">
                    <div class="grid-items">グリッドアイテム1</div>
                    <div class="grid-items">グリッドアイテム2</div>
                    <div class="grid-items">グリッドアイテム3</div>
                    <div class="grid-items">グリッドアイテム4</div>
                    <div class="grid-items">グリッドアイテム5</div>
                    <div class="grid-items">グリッドアイテム6</div>
                </div>
                            
                            
                /* css */
                .grid-wrapper {
                    display: grid;
                }

2. grid ラッパー向け：grid-template プロパティ3種

  1. 縦方向（列）を定義する grid-template-columns プロパティ：グリッド列のライン名とトラックのサイズ変更機能を定義します。
  2. 横方向（行）を定義する grid-template-rows プロパティ
  3. 名前つきのグリッド領域を指定する grid-template-areas プロパティ

2-1. grid-template-columns: none; の場合（初期値）

明示的なグリッドがないことを示します。
どの列も暗黙的に生成され、それらのサイズは grid-auto-columns プロパティで決定される。

                    grid-template-columns: none; /* 初期値 */
                }

2-2. grid-template-columns: max-content; の場合

グリッドトラックを占有しているグリッドアイテムの中で、コンテンツ貢献度の最大値を表すキーワード

2-3. grid-template-columns: min-content; の場合

グリッドトラックを占有しているグリッドアイテムの中で、コンテンツ貢献度の最小値を表すキーワード。
日本語や英数字、記号で幅の取り方が変わる。 日本語は単語の区切りがわからないから1文字が最小単位に、英単語混じりだと、単語の中でも一番長いものが min-content になるになるんじゃないかと教わった。

日本語だと小さい「ぁぃぅぇぉゃゅょっ」、英単語に「ハイフネーション」があると、それが min-content になる。
日本語よりも単語が長い英語が優先される？

2-16. grid-template-areas: "a b c"; の場合（初期値）

グリッドアイテムに指定した名前を利用して、グリッド内の配置位置を指定する。

                    display: grid;
                    grid-template-columns: none; /* 初期値 */
                    grid-template-rows:    none; /* 初期値 */
                    grid-template-areas: "a a a e"
                                         "b c c ."
                                         "b d d ."
                                         "f f f .";
                    }
                .grid-wrapper :nth-child(1){ grid-area: a;}
                .grid-wrapper :nth-child(2){ grid-area: b;}
                .grid-wrapper :nth-child(3){ grid-area: c;}
                .grid-wrapper :nth-child(4){ grid-area: d;}
                .grid-wrapper :nth-child(5){ grid-area: e;}
                .grid-wrapper :nth-child(6){ grid-area: f;}
